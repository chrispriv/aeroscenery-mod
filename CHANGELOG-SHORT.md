# Changelog (Short)

Community-maintained fork of AeroScenery  
Based on AeroScenery 1.1.3-beta

---

## v1.1.3-mod.j
- Moving map functionality for Aerofly FS2 / FS4 (UDP)
- Map modes: fixed map, flight tracing, hide working tiles
- Improved moving-map smoothness (interpolation)
- Enhanced tile & location search with OpenStreetMap geocoding

---

## v1.1.3-mod.i
- Improved usability with tooltips and hints
- 10 m elevation data support (USGS via OpenTopography)
- Trees Presets integration (TreesDetection App)
- Batch image-to-TCC conversion for Android (Mobile)

---

## v1.1.3-mod.h
- Fixed multiple map sources (Switzerland, LINZ, Here WeGo)
- Added PowerShell tools for elevation & OSM data downloads
- QGIS Processing Executor integration
- Improved handling of missing and empty image tiles

---

## v1.1.3-mod.g
- Integrated TreesDetection App as optional processing step
- Configurable parallel downloads (up to 8)
- Tile search by FS2 grid coordinates
- New toolbar shortcuts (User Folder, Scenery Editor)

---

## v1.1.3-mod.f
- PowerShell script for manual OSM data download
- Clipboard helpers for coordinates and grid squares
- Optimized download logic (missing tiles only)

---

## v1.1.3-mod.e
- Added Mapbox map source
- Improved download reliability and retry handling

---

## v1.1.3-mod.d
- ArcGIS map source fixed (minor known issues remain)

---

## v1.1.3-mod.c
- PowerShell script generation for manual image tile downloads

---

## v1.1.3-mod.b
- Added Carto DB Light map source

---

## v1.1.3-mod.a
- Fixed fscloudport airport compatibility issues
